做赛题时遇到的问题

  • AO1 AO2无法满量程 4068

  • display 一次时间太久

    • 分成Init 和刷新部分 且每次刷新屏幕一行

  • IT.c 文件复制模板

    • Get

  • RCC_PCLK1Config（）是什么东西

    • 预分频

  • ADC_GetFlag ADC_FLAG_EOC 完成标志位

    • 1为完成 ！= SET

  • Snprintf对浮点数的对齐处理似乎有问题

    • 手动对齐

  • R37

    • ADC12 CH8

  • RTC ！！！！

    • 注意！要在官方库文件里面加SetCounter！还要修改分频系数！
    • 分频系数要改，然后还要加一个预装载值

  • USART2的RX DMA CH6

    • USART2 RX
    • DMA_SetCurrDataCounter 要先关闭 DMA 使能时才能调用
    • 注意数组的越界！！！！！

  • 关于TIM2的CH3与USART2的TX冲突的解决办法

    • 关闭USART2使能 关闭USART2时钟 使能CH3 ———-&//使能
    • 关闭CH3 打开USART2时钟 打开USART2使能 ———-&//使能

  • define （x）

    • 只是简单代换注意括号

  • usart例程没开 usart 时钟？

    • 在.h文件里面

  • usart与捕获不冲突

    • 对 不冲突

  • 相位不同的互补方波

    • 重置定时器
    • 再重新Init

  • 强制为 高/低 电平 TIM_ForcedOCxConfig

    • 强制电平是类似于 OC 的一种 Mode 用完了需要 TIM_OCxInit

  • 操作LED后要将573的N_LE引脚拉低 这样才能避免操作LCD时影响到LED的亮灭。

  • USART_IT_IDLE 空闲中断要用 USART_ReceiveData(USARTx); 或者 USARTx->SR; USARTx->DR; 清除中断。USART_ClearITPendingBit清除不了。
